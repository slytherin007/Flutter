# Basic Programming Fundamentals Code Samples

This repository contains files that illustrate fundamental programming concepts such as control flow statements, variables, loops, and common data structures like sets, maps, and lists. Each code file provides examples and explanations for these concepts.

## Content Overview

- **Control Flow Statements**: Illustrations of if-else statements, switch statements, and loops for managing program flow.
- **Variables**: Examples showcasing the declaration and usage of variables across different programming languages.
- **Loops**: Demonstrations of loop structures like for, while, and do-while loops.
- **Data Structures**:
  - **Set**: Code snippets exhibiting the implementation and operations on sets.
  - **Map**: Examples showcasing maps or dictionaries and their functionalities.
  - **List**: Illustrations of lists or arrays and their manipulation in various scenarios.
